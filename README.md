# Use the following code to run the node

```
npm install
```

- do the test

```=javascript
npm test problem-??.test
```

- watch the test

```=javascript
npm run test-watch problem-00.test
```

- create an zip for submission

```=javascript
npm run prepare-submission
```

## Version
